Gestor de incidencias

Una API REST diseñada para la gestión de incidencias en un entorno de trabajo en equipo. Las incidencias se registran con un título, una descripción , una clasificación de severidad, etc. La API proporciona un conjunto de endpoints para crear, listar, filtrar según la severidad o el estado, modificar y eliminar incidencias. Esta API facilita la administración eficiente de las incidencias en el equipo de trabajo, permitiendo un seguimiento detallado y personalizado de las mismas. {{msgLink}}

Ecommerce

API REST creada para un entorno de comercio electrónico ofrece una amplia gama de capacidades para gestionar productos, categorías, ventas y usuarios. Sus funcionalidades abarcan desde la visualización y búsqueda de productos y categorías, hasta la creación, edición y eliminación de los mismos. Además, esta API cuenta con un panel de ventas que permite el seguimiento detallado de las transacciones realizadas. Todo ello se complementa con una gestión de usuarios y roles para garantizar un control eficaz de los permisos y el acceso en la plataforma. {{msgLink}}
